My memories

Descripción

Este proyecto fue desarrollado con la finalidad de tener una galería personal en donde poder guardar mis fotos o imágenes de mi agrado teniendo un diseño bastante agradable a la vista, desarrollado utilizando PHP y MySQL. Para la conexión con al base de datos utilizando conceptos de PDO (PHP Data Objects - Objetos de Datos de PHP).
