Aquí están algunas de mis principales destrezas:

Skill 01

PHP

Tengo conocimientos básicos en PHP, lo que me permite manejar tareas simples de desarrollo web, como la creación y manejo de formularios, la interacción con bases de datos MySQL para realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y la integración de PHP con HTML para generar contenido dinámico.

Git+Github

Conocimientos básico en Git y GitHub, lo que me permite gestionar el control de versiones en proyectos de desarrollo. Soy competente en la creación y manejo de repositorios, el uso de ramas para el desarrollo de nuevas funcionalidades. Además, utilizo GitHub para alojar y documentar proyectos.

FLUTTER

Conozco los conceptos fundamentales de Flutter, como el diseño de widgets, la gestión del estado, la navegación entre pantallas y la integración de API.

FIGMA

Mis conocimientos en Figma son básicos, como la creación de diseños. He utilizado Figma en proyectos de diseño de diversos tamaños, lo que me ha permitido comprender un poco su flujo de trabajo y maximizar su potencial para crear experiencias de usuario excepcionales.

HTML+CSS

Puedo crear estructuras semánticas y accesibles para el contenido de las páginas web, mientras que con CSS, puedo dar estilo y diseño a estas estructuras. Estoy familiarizado con las últimas especificaciones de HTML5 y CSS3 y me permite transformar conceptos de diseño en sitios web visualmente atractivos y funcionales.
